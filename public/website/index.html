Key Features

Visual Workflow Builder

Create and visualize workflow templates using an intuitive node-based editor powered by Vue Flow.

Template Management

Design reusable workflow templates with customizable steps and dependencies for any process.

Flow Instances

Generate workflow instances from templates and track their progress in real-time.

Team Assignment

Assign workflows and individual steps to specific teams for better collaboration.

User Management

Comprehensive user and team administration with secure authentication and role-based access.

Email Notifications

Automated email notifications for workflow updates, assignments, and important events.

Audit Logging

Track all system activities and changes with comprehensive audit logs.

PostgreSQL Database

Robust data persistence with Drizzle ORM for reliable and scalable data management.

Responsive UI

Modern Vue 3 interface with Tailwind CSS.

Get Started in Minutes

Docker Compose (Recommended)
git clone https://github.com/kleineLoesungen/flowy
cd flowy
cp .env.example .env
# Edit .env with your settings
docker-compose up -d
Docker Pull
docker pull kleineLoesungen/flowy:latest
docker run -d --name flowy_app \
  -p 3000:3000 \
  --env-file .env \
  kleineLoesungen/flowy:latest
